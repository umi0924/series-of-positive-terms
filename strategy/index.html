    正项级数敛散性：判别法决策流程

    如果你对于收敛性的判断没有思路，以下可以作为一些参考

    步骤 1：第 n 项发散判别法 (N-th Term Test)

    这是所有级数判定的前置条件，用于快速排除明显发散的级数。

    • 执行条件： 计算极限 \(\lim_{n \to \infty} a_n\)。
    • 判定分支：
      • If \(\lim_{n \to \infty} a_n \neq 0\) 或极限不存在：结论为级数发散。判定结束。
      • ElseIf \(\lim_{n \to \infty} a_n = 0\)：该判别法失效。进入步骤 2。

    注意

    \(\lim_{n \to \infty} a_n = 0\) 仅为收敛的必要条件，绝不能以此得出收敛的结论（如调和级数）。

    步骤 2：结构特征匹配 (比值与根值判别法)

    当级数通项包含增长速度极快的结构时，优先使用处理乘除法的高阶判别法。

    === "包含阶乘或常数指数 (比值法)" * 特征提取： 通项 \(a_n\) 中含有 \(n!\) 或 \(c^n\)（如 \(2^n, e^n\)），且通常以乘除形式组合。 * 执行方法： 使用 比值判别法 (Ratio Test)。 * 计算目标： 求 \(\rho = \lim_{n \to \infty} \frac{a_{n+1}}{a_n}\)。

    === "整体 n 次幂结构 (根值法)" * 特征提取： 通项 \(a_n\) 可以写成 \((f(n))^n\) 的形式。 * 执行方法： 使用 根值判别法 (Root Test)。 * 计算目标： 求 \(\rho = \lim_{n \to \infty} \sqrt[n]{a_n}\)。

    步骤 2 判定准则

    • 若 \(\rho < 1\)，级数收敛。
    • 若 \(\rho > 1\) 或 \(\rho = \infty\)，级数发散。
    • 若 \(\rho = 1\)，判别法失效。进入步骤 3。

    步骤 3：渐进分析与连续化 (比较与积分判别法)

    当步骤 2 失效（通常因为通项只包含多项式、有理函数或对数函数，导致 \(\rho=1\)）时，采用以下方法：

    === "有理函数或代数函数 (极限比较法)" * 特征提取： \(a_n\) 由 \(n\) 的多项式或带根号的代数式构成。 * 执行方法： 使用 极限比较判别法 (Limit Comparison Test)。 * 操作步骤： 1. 提取 \(a_n\) 分子与分母的最高次幂，构造等价的简单级数 \(b_n\)（通常为 \(p\)-级数 \(\frac{1}{n^p}\)）。 2. 计算极限 \(c = \lim_{n \to \infty} \frac{a_n}{b_n}\)。 * 判定准则： 若 \(0 < c < \infty\)，则 \(\sum a_n\) 与 \(\sum b_n\) 敛散性相同。

    === "含界函数 (直接比较法)" * 特征提取： 通项中含有易于放缩的有界函数（如 \(\sin^2 n, \cos^2 n, \arctan n\)）。 * 执行方法： 使用 直接比较判别法 (Direct Comparison Test)。 * 操作步骤： 通过不等式放大或缩小，寻找已知的收敛或发散级数作为参照。

    === "含对数或易积分结构 (积分法)" * 特征提取： 通项含有 \(\ln n\)，或整体形式极易求原函数（如 \(\frac{1}{n\ln n}, \frac{n}{n^2+1}\)）。 * 执行方法： 使用 积分判别法 (Integral Test)。 * 操作步骤： 构造对应的连续函数 \(f(x)\)，要求 \(f(x)\) 在 \([1, \infty)\) 上连续、非负且递减。 * 判定准则： 级数 \(\sum a_n\) 与反常积分 \(\int_1^\infty f(x)dx\) 同敛散。

    判别法选择速查表

    级数通项 \(a_n\) 的代数特征 推荐使用的判别法 备注
    极限明显不趋于 \(0\) 第 n 项发散判别法 最优先检查
    含有 \(n!\) 或 \(c^n\) 比值判别法 计算相邻项之比
    整体形式为 \((...)^n\) 根值判别法 开 \(n\) 次方降阶
    两个多项式之商 \(\frac{P(n)}{Q(n)}\) 极限比较判别法 与 \(p\)-级数 \(\sum \frac{1}{n^p}\) 比较
    含有基本有界函数 直接比较判别法 利用不等式进行放缩
    形式易于连续化求积分 积分判别法 注意验证单调递减条件
